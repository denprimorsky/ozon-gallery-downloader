Скрипт для скачивания карточек товаров из галереи на Ozon прямо в выбранную папку на компьютере.

## Установка

1. Установите расширение **[Tampermonkey][tampermonkey-link]**
2. **[Установите скрипт][ozon-gallery-downloader-dist]**
3. Включите режим разработчика в браузере по ссылке chrome://extensions
4. В этом же окне найдите Tampermonkey, выберите его и включите **«Разрешить пользовательские скрипты»** (или «Разрешить доступ к файлам», если есть такой пункт) для корректного отображения меню.
   > Если у вас установлена обычная версия (не Бета), то это будет тут:<br>
   > chrome://extensions/?id=dhdgffkkebhmkfjojejmpbldmpobfkfo

## Как пользоваться

1. Откройте страницу любого товара на [Ozon][ozon-site]
2. Пролистайте галерею, чтобы подгрузились все карточки (работает только при открытии карточек на весь экран)
3. Нажмите на иконку Tampermonkey → выберите `Загрузчик карточек Ozon` → `📥 Скачать карточки в папку`
4. Выберите папку для сохранения в появившемся окне проводника.

> **Важно:** При первом запуске может появиться окно безопасности Tampermonkey с запросом доступа к стороннему ресурсу для загрузки изображений.<br>
> Выберите пункт **«Всегда разрешать для домена»**, чтобы окно не появлялось при следующих скачиваниях.

5. Дождитесь окончания загрузки — статус процесса отображается в правом нижнем углу экрана.

Карточки сохранятся в подпапку с названием вида `Название товара (артикул)`.

> При повторном сохранении в ту же папку подтверждение появляться не будет! Скрипт запоминает папки, для которых вы дали разрешение.

## Возможности

* ✅ Скачивает все видимые карточки из галереи товара
* ✅ Сохраняет изображения в максимальном качестве
* ✅ Создаёт удобную папку с названием и артикулом товара
* ✅ Показывает прогресс и результат в плавающем уведомлении
* ✅ Пропускает дубликаты и битые ссылки
* ✅ При повторном запуске перезаписывает файлы без лишних вопросов

## Настройка

В начале скрипта есть две переменные, которые можно изменить под свои задачи:

```javascript
const DOWNLOAD_DELAY_MS = 250;   // Задержка между загрузками (мс)
const NAME_LIMIT_CHARS = 40;     // Макс. длина названия товара в имени файла
```

* **`DOWNLOAD_DELAY_MS`** — пауза между скачиванием каждой карточки. Если браузер начинает тормозить или блокирует запросы, увеличьте значение (например, до `500` или `1000`).
* **`NAME_LIMIT_CHARS`** — лимит символов для названия товара в имени файла. Если название длиннее, оно обрезается по запятой или последнему пробелу до этого лимита.

Чтобы изменить настройки: откройте панель Tampermonkey → найдите скрипт → нажмите «Редактировать» → поправьте значения вверху файла → сохраните (`Ctrl+S`).

## Требования

* Браузер на движке Chromium: **Chrome**, **Edge**, **Yandex Browser** и т.п.
* Расширение **Tampermonkey** или совместимый менеджер скриптов
* Доступ к [File System Access API](https://developer.mozilla.org/en-US/docs/Web/API/File_System_Access_API) — поэтому в Firefox и Safari не работает

## Примечание

* Скрипт работает только на страницах товаров `https://www.ozon.ru/product/*`
* Карточки сохраняются в формате JPG
* Если галерея не прогружена — пролистайте её вручную перед запуском
* При отмене выбора папки загрузка не начнётся
